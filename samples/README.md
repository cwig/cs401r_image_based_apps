## Image to grayscale

#### OpenCV

```
python other/convert_to_grayscale_opencv.py path/to/input/image.jpg path/to/output/image.jpg
```

#### Numpy / Scipy
```
python other/convert_to_grayscale_numpy_scipy.py path/to/input/image.jpg path/to/output/image.jpg
```
